# 🚀 Loyalty Points Program (Soroban Smart Contract)

## 📌 Project Description

This project is a simple Loyalty Points Program built using Soroban smart contracts on the Stellar blockchain. It allows businesses or applications to reward users with points and enables users to redeem those points securely.

---

## ⚙️ What It Does
- Stores loyalty points for each user on-chain
- Allows adding points to a user account
- Enables users to check their balance
- Supports redeeming points

---

## ✨ Features
- ✅ Decentralized storage of loyalty points
- ✅ Secure and transparent transactions
- ✅ Simple and efficient contract logic
- ✅ User-based point tracking using blockchain addresses
- ✅ Prevents overdraft (cannot redeem more than available points)

---

## 🛠️ Functions Overview

### 1. Add Points
Adds loyalty points to a user's account.

### 2. Get Points
Fetches the current points balance of a user.

### 3. Redeem Points
Allows users to redeem points if they have sufficient balance.
